\# Lendrix Public API



This repository defines the public API layer of the Lendrix platform.



It serves as the canonical reference for:

\- external integrations

\- SDK development

\- partner systems

\- ecosystem services

\- public verification endpoints

\- proof and transparency systems



This is a specification repository, not an implementation.
